Detailed Description

template<typename T, size_t SIZE = 1000, size_t K = 5>
class set::filters::BloomFilter< T, SIZE, K >

Class that implements a BloomFilter for O(1) check if an element is NOT present into the Set. Query could return false-positive, that's why in the event the element is found it returns MAYBE.

Parameters
SIZEthe size of the bloomfilter array
Kthe number of hashing functions
