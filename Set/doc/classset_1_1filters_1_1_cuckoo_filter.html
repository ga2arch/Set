Detailed Description

template<typename T, size_t SIZE = 100, size_t BUCKETS = 4, size_t MAX_DEPTH = 100>
class set::filters::CuckooFilter< T, SIZE, BUCKETS, MAX_DEPTH >

Class to implement a CuckooFilter, a lot similar to the Cuckoo Hash Table, but instead of storing the element, it stores the fingerprint (just another hash) and makes use of partial hashing for easily retrieve the second hash from the first.

Parameters
SIZEthe size of the filter
BUCKETSthe number of buckets to use.
MAX_DEPTHdepth cutoff in case of eviction.
